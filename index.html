Javascript Variables

What is a variable?

A variable is an identifier or a name which is used to refer a value . A variable is written with a combination of letters numbers and special characters_(underscore) and $(dollar) provided that the first letter must be an alphabet.
Ex: d, fact, b11, total_amount etc.


Rules for naming a variable.

  • Variable can contain combination of letters, digits, underscores(_), and dollar signs ($).
  • Must begin with a letter A-z or a-z or underscore or dollar sign.
  • A variable name cannot start with a number.
  • Must not contain any space characters.
  • JavaScript is case sensitive.
  • Can't use reserved keywords.


    • Declaring a variable

    Syntax: scope variable_name;
    Example: var a;
    Here var is the scope a is the variable name. The above satetment means that a variable is declared of scope "var" and variable name "a" which is refering to a undefined value or a garbage value.

    Initialization of a variable

    Syntax: scope variable_name = value ;
    Example: var a = 5; or var a = "js" ;
    Here var is the scope a is the variable name. The above satetment means that a variable is declared of scope "var" and variable name "a" which is refering to a 5 or "js" respectively.

    Note: The scope of the variable can be omitted, it can also be written as: a = 5;

    Important Points

  • Variable can contain combination of letters, digits, underscores(_), and dollar signs ($).
  • Strings are written inside double or single quotes.
  • Number are written without quotes.
  • If you put a number in quotes, it will be treated as a text string.

    • Initialization of a variable in a line .

    Example: var a = 5, b = 10 , c = 10;
                      var a = "js", b = 10 , c = "10";
